# SmoothStep2D Framework sample applications

Sample games of [SmoothStep2D Framework](http://ss2d.wordpress.com).


## Installation

    git clone https://github.com/sodesign/ss2d_samples.git

    cd ss2d_samples

    cd XX_SAMPLENAME

    npm install

    node app.js
